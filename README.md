<div align="center">
  <h1>Bike Manufacturing Factory</h1>
</div>

This repository contains a desktop-based application that was developed using PyQt for a fictional bike manufacturing company. The application was designed to digitalise the factory's manufacturing workflow, allowing staff to monitor bicycle production from start to completion.

The system integrates various aspects of bike production into a single interface - supporting workflow monitoring, inventory management, part purchasing from suppliers and customer ordering management. Moreover, graphs are featured to visualise production over time, enabling potential decision-makers to observe trends and better match supply to demand.

The Model-View-Controller (MVC) pattern is employed to efficiently separate the user interface, business logic and data management, improving the maintainability of the application. Core functionality of the system's business logic is verified using Python’s unittest framework.
